# Logos RFP Program

The Logos RFP Program funds and accelerates production-ready applications built by external teams that showcase the capabilities of the Logos Stack.

[![Submit Proposal](https://img.shields.io/badge/Submit%20Proposal-Open%20Issue-blue)](https://github.com/<OWNER>/<REPO>/issues/new?template=proposal.yml)

---

## How it works

1. Browse the open RFPs below
2. Read the RFP you want to apply for in the `rfps/` folder
3. Submit a proposal using the button above
4. We review proposals weekly and typically respond within **14 days**

Use GitHub Discussions for questions and clarifications.

---

## What we fund

Proposals should align with one or more focus areas:

- **Developer Tooling and Infrastructure**
- **Applications and Integrations**
- **Ecosystem and Community Enablement**

All accepted technical deliverables must be open-sourced under the **MIT License** unless explicitly approved otherwise.

## How to add a new RFP

1. Create a new file in `rfps/` named `RFP-###-short-title.md`
2. Include YAML frontmatter at the top like:

```yaml
---
id: RFP-001
title: Example Title
tier: M
funding: $5,000–$15,000
status: open
category: Applications and Integrations
---
